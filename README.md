## 服務說明

這是個可以讓人隨時上傳或搜尋他人製作的調酒酒譜平台

## 使用方式

1. 可註冊及登入會員
2. 新增分享、移除自己的酒譜
3. 可以搜尋自己的酒譜

 * Create Recipe
 * Recipe List
 * Detail Recipe
 * Delete Recipe
 * Search Recipe
 * User login, sign up, logout
 * Personal Page

## 使用之框架/模組/程式碼
```
Frontend
    - React
    - React-router-dom
    - axios
    - Material UI

Backend
    - Express
    - Cors
    - jwt.io
    - Bcrypt

Database
    - mongoDB
```
